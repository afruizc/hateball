
Ready to run in production? Please [check our deployment guides](https://hexdocs.pm/phoenix/deployment.html).

  
## Deployment

To deploy run (on the machine where it's being deployed):

```bash
./scripts/clean.sh
MIX_ENV=prod ./scripts/build.sh
./scripts/run.sh
```
